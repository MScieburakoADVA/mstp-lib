STP_GetPortAutoEdge

bool STP_GetPortAutoEdge
(
    const STP_BRIDGE*  bridge,
    unsigned int       portIndex,
);

Summary

Gets the AutoEdge variable for a given port.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port.

Remarks

See §13.27.18 in 802.1Q-2018 for more information.

The STP Library initializes this variable to TRUE, as recommended in the standard.

This function may be called from within an STP callback.
