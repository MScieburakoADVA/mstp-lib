STP_OnPortDisabled

void STP_OnPortDisabled
(
    STP_BRIDGE*   bridge,
    unsigned int  portIndex,
    unsigned int  timestampMilliseconds
);

Summary

Function which the application must call when the operational state of a port changes to FALSE. See the Remarks section of this page for more details, and also the Remarks section of STP_OnPortEnabled.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port whose operational state has just changed to FALSE.
timestamp
A timestamp used for the debug log.

Remarks

Also see the Remarks section of STP_OnPortEnabled.

It is allowed to call this function for already disabled ports.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
