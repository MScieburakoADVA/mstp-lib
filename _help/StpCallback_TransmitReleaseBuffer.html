StpCallback_TransmitReleaseBuffer

void StpCallback_TransmitReleaseBuffer
(
    const STP_BRIDGE* bridge,
    void* bufferReturnedByGetBuffer
);

Summary

Used together with StpCallback_TransmitGetBuffer to transmit a BPDU generated by the STP library.

Parameters

bridge
The application receives in this parameter a pointer to the bridge object returned by STP_CreateBridge.
bufferReturnedByGetBuffer
The application receives in this parameter the same value it returned from StpCallback_TransmitGetBuffer. The callback implementation may or may not need this value.

Remarks

This callback is called by the STP library only if StpCallback_TransmitGetBuffer returned a non-NULL address.
