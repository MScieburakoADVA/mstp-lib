STP_StopBridge

void STP_StopBridge
(
    STP_BRIDGE*   bridge,
    unsigned int  timestamp
);

Summary

Stops a bridge (i.e., stops executing the bridge's state machines).

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
timestamp
A timestamp used for the debug log.

Remarks

It is allowed to call this function for already-stopped bridges.

The current library code was not tested in scenarios where a bridge is stopped with this function.

This function may not be called from within an STP callback.
