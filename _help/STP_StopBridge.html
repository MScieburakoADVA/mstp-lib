STP_StopBridge

void STP_StopBridge
(
    STP_BRIDGE*   bridge,
    unsigned int  timestamp
);

Summary

Stops the Spanning Tree Protocol.

Parameters

bridge
Pointer to an STP_BRIDGE object, obtained from STP_CreateBridge.
timestamp
A timestamp used for the debug log.

Remarks

This function calls StpCallback_EnableBpduTrapping to restore the switch to its default, non-STP-capable, state.

This function re-enables learning and forwarding. It does so by calling StpCallback_EnableLearning and StpCallback_EnableForwarding for every tree of every port that had either of them disabled.

This function may not be called from within an STP callback.
