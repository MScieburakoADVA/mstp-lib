STP_SetStpVersion

void STP_SetStpVersion
(
    STP_BRIDGE*       bridge,
    enum STP_VERSION  version
);

Summary

Sets the STP version of the bridge, referenced to as "ForceProtocolVersion" in the STP standard.

Parameters

bridge
Pointer to a STP_BRIDGE object.
protocolVersion
One of the STP_VERSION values.

Remarks

This function is currently not implemented. STP version is set with STP_CreateBridge and can't be changed afterwards.

See §13.22.e) in 801.1Q-2011 for more information.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
