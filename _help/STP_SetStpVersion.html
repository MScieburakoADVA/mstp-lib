STP_SetStpVersion

void STP_SetStpVersion
(
    STP_BRIDGE*       bridge,
    enum STP_VERSION  version
);

Summary

Sets the STP version of the bridge, referred to as ForceProtocolVersion in the STP standard.

Parameters

bridge
Pointer to an STP_BRIDGE object.
protocolVersion
One of the STP_VERSION values.

Remarks

See §13.7.2 in 801.1Q-2018 for more information.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
