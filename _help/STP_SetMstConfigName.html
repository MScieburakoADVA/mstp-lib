STP_SetMstConfigName

void STP_SetMstConfigName
(
    STP_BRIDGE*   bridge,
    const char*   name,
    unsigned int  timestamp
);

Summary

Sets the Name field of the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.
name
The new name. This must be a null-terminated string of up to 32 characters (up to 33 with the null-terminator).
timestamp
A timestamp used for the debug log.

Remarks

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.8 in 802.1Q-2018 for more information about the MST Configuration Identifier.

If this function is not called, a default configuration name is used which is generated by STP_CreateBridge. See the Remarks section there for more information.

This function may not be called from within an STP callback.
