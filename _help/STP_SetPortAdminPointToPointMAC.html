STP_SetPortAdminPointToPointMAC

void STP_SetPortAdminPointToPointMAC
(
    STP_BRIDGE*    bridge,
    unsigned int   portIndex,
    STP_ADMIN_P2P  adminPointToPointMAC
);

Summary

Sets the adminPointToPointMAC variable for a given port.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port.
adminPointToPointMAC
One of the STP_ADMIN_P2P values.

Remarks

See §6.6.3 in 802.1Q-2011 for more information.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
