STP_GetTreeIndexFromVlanNumber

unsigned int STP_GetTreeIndexFromVlanNumber
(
    STP_BRIDGE*   bridge,
    unsigned int  vlanNumber
);

Summary

Retrieves a tree index mapped to a given VLAN.

Parameters

bridge
Pointer to a STP_BRIDGE object.
vlanNumber
The VLAN number. This must be between zero (inclusive) and the maxVlanNumber parameter that was passed to STP_CreateBridge (inclusive).

Return value

The tree index mapped to the specified VLAN.

Remarks

You can call this function from within an STP callback.
