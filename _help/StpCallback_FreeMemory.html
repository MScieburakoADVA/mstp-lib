StpCallback_FreeMemory

void StpCallback_FreeMemory (void* p);

Summary

Application-defined function that frees memory previously allocated with StpCallback_AllocAndZeroMemory.

StpCallback_FreeMemory is a placeholder name used throughout this documentation. The application may name this callback differently.

Parameters

p
The application receives in this parameter a pointer to the memory block to be freed.
