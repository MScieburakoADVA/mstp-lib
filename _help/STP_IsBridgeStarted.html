STP_IsBridgeStarted

bool STP_IsBridgeStarted (STP_BRIDGE* bridge);

Summary

Checks whether a bridge is started or stopped.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.

Return value

true if the bridge is started, false if the bridge is stopped.

Remarks

A bridge can be started by calling STP_StartBridge, and stopped by calling STP_StopBridge.

It is allowed to call this function from within an STP callback.
