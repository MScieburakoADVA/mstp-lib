STP_GetMstConfigTable

void STP_GetMstConfigTable
(
    STP_BRIDGE*    bridge,
    unsigned char  mstidsOut [4094]
);

Summary

Retrieves a table containing the mapping of VLANs to MSTIs.

Parameters

bridge
Pointer to a STP_BRIDGE object.
mstidsOut
Pointer to an array of 4094 bytes where this function will write the mapping.

Remarks

Element at index 0 of the array corresponds to VLAN 1, element at index 1 corresponds to VLAN 2, and so on until the element at index 4093 which corresponds to VLAN 4094.

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.7 in 802.1Q-2011 for more information about the MST Configuration Identifier.

It is allowed to call this function from within an STP callback.
