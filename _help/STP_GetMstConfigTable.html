STP_GetMstConfigTable

const struct STP_CONFIG_TABLE_ENTRY* STP_GetMstConfigTable
(
    const STP_BRIDGE*  bridge,
    unsigned int*      entryCountOut
);

Summary

Retrieves a pointer to the MST configuration table (the VLAN to tree mapping).

Parameters

bridge
Pointer to an STP_BRIDGE object.
entryCountOut
Pointer to a variable where this function writes the number of entries available. This is maxVlanNumber + 1, where maxVlanNumber is the parameter that was passed to STP_CreateBridge.

Return Value

The function returns a pointer to the table kept internally by the library.

Remarks

You can call this function even when the bridge is not running MSTP.

Element at index 0 of the table is unused and contains the value 0. Element at index 1 corresponds to VLAN 1, element at index 1 corresponds to VLAN 2 and so on. The table can contain up to 4096 elements.

See §13.8 in 802.1Q-2018 for more information about the MST Configuration Identifier.

It is allowed to call this function from within an STP callback.

See also STP_SetMstConfigTable.
