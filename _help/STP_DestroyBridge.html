STP_DestroyBridge

void STP_DestroyBridge (STP_BRIDGE* bridge)

Summary

Releases all memory allocated for the given bridge.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.

Remarks

The bridge must be stopped when this function is called (i.e., must have never been started, or must have been stopped with STP_StopBridge).

This function may not be called from within an STP callback.
