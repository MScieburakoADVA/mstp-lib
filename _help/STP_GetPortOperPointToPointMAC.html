STP_GetPortOperPointToPointMAC

bool STP_GetPortOperPointToPointMAC
(
    const STP_BRIDGE*  bridge,
    unsigned int       portIndex,
);

Summary

Gets the operPointToPointMAC variable for a given port.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port.

Remarks

See §11.3 in 802.1AC-2016 for more information.

This function may be called from within an STP callback.
