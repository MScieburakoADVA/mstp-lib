STP_SetMstConfigRevisionLevel

void STP_SetMstConfigRevisionLevel (STP_BRIDGE* bridge, unsigned short revisionLevel);

Summary

Sets the Revision Level field of the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.
revisionLevel
The new Revision Level value.

Remarks

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.8 in 802.1Q-2018 for more information about the MST Configuration Identifier.

If this function is not called, a default revision level is used which is generated by STP_CreateBridge. See the Remarks section there for more information.

This function may not be called from within an STP callback.
