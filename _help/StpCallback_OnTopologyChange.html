StpCallback_OnTopologyChange

void StpCallback_OnTopologyChange (STP_BRIDGE* bridge, unsigned int treeIndex);

Summary

Application-defined function that is called by the STP library when it detects a change in the network topology.

Parameters

bridge
The application receives in this parameter a pointer to a STP_BRIDGE object.
treeIndex
The index of the spanning tree on which a topology change occurred.

Remarks

The STP library calls this callback when it detects a topology change on the local bridge.
