STP_GetPortPriority

unsigned char STP_GetPortPriority
(
    const STP_BRIDGE*  bridge,
    unsigned int       portIndex,
    unsigned int       treeIndex
);

Summary

Reads the priority component - most significant 4 bits - of the portId. See §13.25.32 in 802.1Q-2011 for more details.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The zero-based index of the port.
treeIndex
The zero-based index of the spanning tree for which the port priority is to be read. For STP or RSTP, this must be zero. For MSTP, this must be zero for the CIST, or 1..64 for an MSTI.

Return value

The port priority. This is a number between 0 and 244, in steps of 16.

Remarks

It is allowed to call this function from within an STP callback.
