STP_GetBridgePriority

unsigned short STP_GetBridgePriority
(
    const STP_BRIDGE*  bridge,
    unsigned int       treeIndex
);

Summary

Reads the bridge priority, that is, the priority component - most significant 4 bits - of the BridgeIdentifier (§13.24.1 in 802.1Q-2011).

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
treeIndex
The zero-based index of the spanning tree for which the bridge priority is to be returned. For STP or RSTP, this must be zero. For MSTP, this must be zero for the CIST, or 1..64 for an MSTI.

Return value

The bridge priority. This is a number between 0 and 61440 (0xF000) hex, in steps of 4096 (0x1000 hex).

Remarks

See the Remarks section at STP_SetBridgePriority for more information about the bridge priority and about the BridgeIndentifier.

It is allowed to call this function from within an STP callback.
