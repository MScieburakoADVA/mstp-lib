STP_GetMstConfigRevisionLevel

unsigned short STP_GetMstConfigRevisionLevel (STP_BRIDGE* bridge);

Summary

Gets the Revision Level field of the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.

Return value

The Revision Level value.

Remarks

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.7 in 802.1Q-2011 for more information about the MST Configuration Identifier.

It is allowed to call this function from within an STP callback.
