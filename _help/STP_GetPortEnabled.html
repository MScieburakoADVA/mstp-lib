STP_GetPortEnabled

bool STP_GetPortEnabled
(
    const STP_BRIDGE*  bridge,
    unsigned int       portIndex
);

Summary

Reads the portEnabled variable (§13.25.31 in 802.1Q-2011).

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The zero-based index of the port.

Return value

This function returns true if STP_OnPortEnabled was called last. It returns false if STP_OnPortDisabled was called last, or the bridge was just created with STP_CreateBridge.

Remarks

It is allowed to call this function from within an STP callback.
