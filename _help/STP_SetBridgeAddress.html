STP_SetBridgeAddress

void STP_SetBridgeAddress
(
    STP_BRIDGE*           bridge,
    const unsigned char*  address,
    unsigned int          timestamp
);

Summary

Function which the application must call every time the bridge's MAC address changes.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
address
The new MAC address of the STP bridge. See the Remarks section for important information about this MAC address.
timestamp
A timestamp used for the debug log.

Remarks

The MAC address of the bridge is allowed to be different from the MAC address used by other Ethernet protocols running on the same device.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

See the Remarks section in STP_SetBridgePriority for information on how the bridge priority is calculated from the bridge address.

This function may not be called from within an STP callback.
