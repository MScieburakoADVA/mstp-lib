STP_GetVersionFromString

enum STP_VERSION STP_GetVersionFromString
(
    const char* str
);

Summary

Converts a string to and STP_VERSION_XXX value.

Parameters

str
Pointer to a null-terminated string.

Return value

An STP_VERSION_XXX value.

Remarks

The function checks with an assertion that the string is one of those returned by STP_GetVersionString.
