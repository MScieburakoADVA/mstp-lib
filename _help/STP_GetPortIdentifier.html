STP_GetPortIdentifier

unsigned short STP_GetPortIdentifier
(
    const STP_BRIDGE*  bridge,
    unsigned int       portIndex,
    unsigned int       treeIndex
);

Summary

Returns the port identifier portId for a given port and a given tree.

Parameters

bridge
Pointer to a STP_BRIDGE object.
portIndex
The zero-based index of the port.
treeIndex
The zero-based index of the spanning tree for which the portId is to be read. For STP or RSTP, this must be zero. For MSTP, this must be zero for the CIST, or 1..64 for an MSTI.

Return value

The port identifier. This is a number between 0 and 244, in steps of 16.

Remarks

See §13.27.46 in 802.1Q-2018 for more details.

It is allowed to call this function from within an STP callback.
