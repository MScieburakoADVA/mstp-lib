StpCallback_EnableLearning

void StpCallback_EnableLearning
(
    const STP_BRIDGE* bridge,
    unsigned int      portIndex,
    unsigned int      treeIndex,
    bool              enable
);

Summary

See StpCallback_EnableForwarding. All information from that callback applies to this callback as well, except that learning is enabled/disabled, not forwarding.

StpCallback_EnableLearning is a placeholder name used throughout this documentation. The application may name this callback differently.
