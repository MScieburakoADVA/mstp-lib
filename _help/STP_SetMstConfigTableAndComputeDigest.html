STP_SetMstConfigTableAndComputeDigest

void STP_SetMstConfigTableAndComputeDigest
(
    STP_BRIDGE*                 bridge,
    const struct VLAN_TO_MSTID* table,
    unsigned int                tableEntryCount,
    unsigned int                timestamp
);

Summary

Sets the MST Config Table (VLAN to MSTID mapping) and computes the Digest field of the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.
table
Pointer to an array of VLAN_TO_MSTID entries, where each entry specifies a mapping between a VLAN and its corresponding MSTID.
tableEntryCount
The number of entries in table.
timestamp
A timestamp used for the debug log.

Remarks

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.7 in 802.1Q-2011 for more information about the MST Configuration Identifier.

The bridge supports only the number of trees that was specified in the call to STP_CreateBridge. The caller must not attempt to map a VLAN to a non-existent tree, i.e., the table passed to this function must contain only mstids lower than the treeCount parameter that was specified in the call to STP_CreateBridge. The STP library will raise an assertion if the caller attempts to do that.

The STP library computes the Digest as described in 13.7.4) in 802.1Q-2011.

If this function is not called, a default mapping is used which is generated by STP_CreateBridge. See the Remarks section there for more information.

This function may not be called from within an STP callback.
