STP_GetPortAdminPointToPointMAC

STP_ADMIN_P2P STP_GetPortAdminPointToPointMAC
(
    STP_BRIDGE*   bridge,
    unsigned int  portIndex,
);

Summary

Gets the adminPointToPointMAC variable for a given port.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port.

Remarks

See §6.6.3 in 802.1Q-2011 for more information.

This function may be called from within an STP callback.
