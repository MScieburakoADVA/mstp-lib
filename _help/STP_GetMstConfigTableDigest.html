STP_GetMstConfigTableDigest

const unsigned char* STP_GetMstConfigTableDigest (STP_BRIDGE* bridge);

Summary

Gets a pointer to the Digest field of the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.

Return value

A pointer to a 16-byte long array of bytes containing the Configuration Digest currently in use.

Remarks

This function may be called only for MSTP bridges (i.e., if STP_VERSION_MSTP was passed to STP_CreateBridge or STP_SetStpVersion).

See §13.7 in 802.1Q-2011 for more information about the MST Configuration Identifier.

The Configuration Digest is initialized by STP_CreateBridge, and recomputed whenever STP_SetMstConfigTableAndComputeDigest or STP_SetMstConfigTableAndComputeDigest1 are called.

It is allowed to call this function from within an STP callback.
