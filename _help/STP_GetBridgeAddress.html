STP_GetBridgeAddress

void STP_GetBridgeAddress
(
    STP_BRIDGE*     bridge,
    unsigned char*  addressOut6Bytes,
);

Summary

Gets the bridge address that was set with STP_CreateBridge or STP_SetBridgeAddress.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
addressOut6Bytes
The MAC address of the STP bridge

Remarks

See the Remarks section in STP_SetBridgePriority for information on how the bridge priority is calculated from the bridge address.

It is allowed to call this function from within an STP callback.
