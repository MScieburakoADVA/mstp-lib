STP_SetPortAutoEdge

void STP_SetPortAutoEdge
(
    STP_BRIDGE*   bridge,
    unsigned int  portIndex,
    bool          autoEdge
);

Summary

Sets the AutoEdge parameter for a given port.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The index of the port for which the AutoEdge parameter is to be set.
newAutoEdge
The new value of the AutoEdge parameter.

Remarks

For more information, see §13.27.18 in 802.1Q-2018.

The STP Library initializes this variable to TRUE, as recommended in the standard.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
