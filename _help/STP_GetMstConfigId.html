STP_GetMstConfigId

const struct STP_MST_CONFIG_ID* STP_GetMstConfigId
(
    const STP_BRIDGE*  bridge,
);

Summary

Retrieves a pointer to the MST Configuration Identifier.

Parameters

bridge
Pointer to a STP_BRIDGE object.
nameOut
Pointer to an array of chars where this function will write the name.

Remarks

You can call this function even while the bridge is not running MSTP.

See §13.7 in 802.1Q-2011 for more information about the MST Configuration Identifier.

It is allowed to call this function from within an STP callback.
