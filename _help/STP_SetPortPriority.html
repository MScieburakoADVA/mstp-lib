STP_SetPortPriority

void STP_SetPortPriority
(
    STP_BRIDGE*    bridge,
    unsigned int   portIndex,
    unsigned int   treeIndex,
    unsigned char  portPriority,
    unsigned int   timestamp
);

Summary

Sets the priority component - most significant 4 bits - of the portId.

Parameters

bridge
Pointer to a STP_BRIDGE object, obtained from STP_CreateBridge.
portIndex
The zero-based index of the port.
treeIndex
The zero-based index of the spanning tree for which the port priority is to be set. For STP or RSTP, this must be zero. For MSTP, this must be zero for the CIST, or 1..64 for an MSTI.
portPriority
A value between 0 and 240, in steps of 16.
timestamp
A timestamp used for the debug log.

Remarks

See §13.25.32 in 802.1Q for more details.

Execution of this function is a potentially lengthy process. It may call various callbacks multiple times.

This function may not be called from within an STP callback.
